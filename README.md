# MeshOff OS

Offline-first Mesh Operating System for field communication.

## 🔥 Features

- BLE + USB LoRa bridge (Meshtastic compatible)
- GPS live tracking
- Terminal UI (green theme)
- Mouse + keyboard support
- Offline map server
- First boot setup wizard
- Fully offline boot system

## 🧠 Designed for

- Raspberry Pi Compute Module 4
- uConsole (ClockworkPi)
- LoRa mesh networks

## 🚀 Quick install

Download latest release:

👉 https://github.com/studioetrange-ux/releases

Flash with Raspberry Pi Imager.

## ⚙️ Default mode

- Offline-first
- Auto hardware detection
- No internet required

## 📡 Hardware

- LoRa device (ESP32 / SX1262)
- GPS module (UART)
